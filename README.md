# rapg

## Installation
from Github
```
git clone https://github.com/kanywst/rapg
cd rapg/cmd/rapg
go build .
mv rapg /usr/local/bin
```
## Usage
```
$ rapg -h
Usage of ./rapg:
  -a    Show All Password.
  -i string
        Set Domain/Username for passsword. (default "null")
  -l int
        Set Password Length (default 20)
  -s string
        Search for Password. (default "null")
```
