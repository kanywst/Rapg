# rapg

## Installation
from Github
```
git clone https://github.com/kanywst/rapg
cd rapg/cmd/rapg
go build .
mv rapg /usr/local/bin
```
## Usage
```
$ rapg -h                                                                                                                                                                                    [master]
Usage of rapg:
  -a    Show All Password.
  -k string
        Set Keyword for passsword. (default "null")
  -l int
        Set Password Length (default 20)
  -s string
        Search for Password. (default "null")
  -u string
        Set Username. (default "null")
```
